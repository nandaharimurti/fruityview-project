🔧 TROUBLESHOOTING:

  ❌ Login tidak berfungsi?
     └─ Cek console browser (F12) untuk error messages
     └─ Verifikasi path ke AuthController sudah benar
     └─ Pastikan database sudah terhubung

  ❌ Notification tidak muncul?
     └─ Cek apakah login.css sudah ter-load
     └─ Verifikasi .notification-toast style ada di CSS

  ❌ Password toggle tidak bekerja?
     └─ Pastikan login.js sudah ter-load
     └─ Cek browser console untuk error

  ❌ Redirect tidak terjadi?
     └─ Verifikasi path redirect di AuthController
     └─ Pastikan home/index.php sudah ada
